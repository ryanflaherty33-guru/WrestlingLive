# ⚡ LIVEWIRE

*Ride the current. Flip the forks.*

You are a spark of electricity racing along an endless neon circuit. The wire
keeps forking ahead of you — **tap anywhere** to flip the switch at the next
fork before you get there.

- Follow the **bright wire**; if it turns **red**, you're routed into a blown
  fuse — tap!
- Grab **charge orbs** to stack your combo multiplier (up to 9x).
- **Triple forks** hide a golden gem jackpot; taps cycle all three paths.
- **Blinking breakers** guard bonus orbs — time your way through.
- Clear enough forks and the circuit levels up: faster current, tighter
  forks, new colors. You get 3 fuses per run.

Post your initials to the **world leaderboard** and settle it.

## Play

The game is a website — open it on your phone and go. It deploys
automatically from `main` via Vercel.

## Tech

Expo / React Native (runs on web via react-native-web), SVG rendering,
synthesized Web Audio sound effects, Supabase-backed global leaderboard.

```bash
npm install
npm run web     # local dev
```
